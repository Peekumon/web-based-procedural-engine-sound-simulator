# Engine Simulator M3 (WBPESS)

A lightweight, browser-based procedural engine simulator built with HTML, CSS and JavaScript.  
It features adjustable engine parameters, turbo simulation, live gauges, dyno visualization, and synthesized engine audio.

## Features

- Adjustable engine parameters (cylinders, displacement, bore, stroke, compression, etc.)
- Turbocharger simulation with lag, spool, boost and intake noise
- Real-time RPM, torque, horsepower, AFR and temperature gauges
- Interactive dyno chart with torque and power curves
- Audio engine using procedural oscillators and noise synthesis
- Multiple engine presets (2JZ, RB26, LS3, F1 V10, EJ25, Diesel)
- Responsive layout with sidebar controls and visual dashboard
- No external libraries required
